From C to C++

Everything you know about C applies, but C++ adds

  • Some new language features (e.g., references, which are easier to use than pointers for some purposes)

  • The ability to define classes and do object-oriented programming (similar to but not same as Java)

  • A better way of representing text strings

  • Different library for I/O

  • Useful libraries for representing and manipulating data structures such as linked lists, maps, etc

C Example

#include <stdio.h>
#include <string.h>

int main()
{
  char name[10];

  printf("Enter your name: ");
  scanf("%s", name);

  printf("\nHello %s!\n", name);

  int num_chars = strlen(name);
  printf("Your name contains %d characters\n", num_chars);

  return 0;
}
1
We must include header files to access standard I/O and string handling functions
2
Strings are represented as char arrays (or pointers to char)
3
printf() is used to do formatting printing to stdout
4
scanf() is used to do formatted reading from stdin (but is error-prone)
5
strlen() tells us length of a string (by counting chars until a null byte)

Equivalent in C++

#include <iostream>
#include <string>

int main()
{
  std::string name;

  std::cout << "Enter your name: ";
  std::cin >> name;

  std::cout << "\nHello " << name << "!\n";

  std::cout << "Your name contains " << name.length()
            << " characters" << std::endl;

  return 0;
}
1
C++ has equivalent headers for I/O & string handling (note: no filename suffix!)
2
C++ has a string class for handling text (similar to Java’s String class)
3
std::cout is the console output stream; << operator is used to insert data into the stream
4
std::cin is the console input stream; >> operator is used to extract data from the stream into a variable
5
We can chain stream insertion (or extraction) operations together
6
length() method tells us the size of a string

std::

  • Standard library features are defined in the std namespace

  • Names of things from that library should normally be qualified with the std:: prefix, but you can avoid this:

#include <iostream>
#include <string>

using namespace std;

int main()
{
  // Can now refer to string instead of std::string,
  // cout instead of std::cout, etc
}
1
Put this after included headers, near top of a .cpp file; do NOT put it in header files!

I/O in C++

  • iostream header gives you access to cin & cout, representing the console input stream (stdin) and console output stream (stdout), respectively

  • fstream header gives you access to classes representing file-based streams, with which you can do file I/O

  • iomanip header gives you access to manipulators that control streams in various ways (e.g., to format numbers)

  • << operator is used to insert data into a stream; >> is used to extract data from a stream into variables

Checking For Success

We can call stream’s fail() method; non-zero return value indicates that something went wrong:

double x, y;
cout << "Enter x & y coordinates: ";
cin >> x >> y;
if (cin.fail()) {
  // Handle error somehow
}

Or we can embed the input operations into an if statement; it will evaluate to true if the operations succeeded

if (cin >> x >> y) {
  // OK to use x & y here because values were
  // read successfully into both variables
}

Better Strings

  • C’s representation of strings as null-terminated char arrays is low-level, hard to use and error-prone

  • C++ strings are much more like those in Java – i.e., they are instances of a string class

  • C++ strings know their size, manage their own storage and have many useful methods, just like Java strings

  • Unlike in Java, C++ strings are mutable

Example

#include <string>
...

string target = "World";
string message = "Hello " + target + "!";

cout << message << endl;
cout << message.length() << endl;
cout << message[6] << endl;
cout << message.find("World") << endl;

message.replace(0, 5, "Goodbye");
cout << message << endl;
1
+ used to concatenate strings, just like Java
2
Prints Hello World!
3
Prints W, the seventh character in message
4
Prints 6, the index of the first occurrence of substring World
5
Replaces characters from index 0 up to (but not including) 5 with Goodbye
6
Prints Goodbye World!

C++ Collection Types

  • Primitive arrays, like C
    • Static allocation, on stack
    • Dynamic allocation, on heap
  • Types in standard library:
    • vector (like Java’s ArrayList)
    • list (like Java’s LinkedList)
    • set (like Java’s HashSet)
    • map (like Java’s HashMap)

Dynamic Storage in C

  • Use malloc() or calloc() functions to allocate heap storage
    • Both return a pointer to that storage
  • Use free() to release storage after we are done with it
float* data = (float*) calloc(100, sizeof(float));
...
free(data);

Dynamic Storage in C++

  • Like Java, use new operator to allocate heap storage
  • Unlike Java, use delete to free storage after we are done
float* data = new float[100];
...
delete[] data;
1
Need the [] only when allocating storage for multiple values of the specified type

Note

C++11 onwards also has smart pointers, which can release storage automatically when pointer goes out of scope

Creating Vectors

#include <vector>
#include <string>
...

vector<int> a;
vector<int> b(10);
vector<int> c(5, -1);
vector<int> d(c);

vector<string> p;
vector<string> q(5, "xyz");
1
a is an empty vector capable of holding int values
2
b is a vector of 10 int values, all 0 (the default)
3
c is a vector of 5 int values, all -1
4
d is a copy of c
5
p is an empty vector capable of holding strings
6
q is a vector of 5 strings, all equal to "xyz"

Note

Element type is specified in angle brackets, as we do for ArrayList in Java

Using Vectors

vector<string> words;

words.push_back("Apple");
words.push_back("Banana");
words.push_back("Kiwi");
cout << words.size() << endl;

words.pop_back();

words.clear();
1
Puts Apple at end
2
Puts Banana after Apple
3
Puts Kiwi after Banana
4
Prints 3
5

6
Also prints Banana, but at() method does bounds checking
7
Removes (and returns) Kiwi

Prints 2

Pointer Parameters

A function can modify values of variables passed to it if we use pointers are parameters:

void swap(int* x, int* y)
{
  int temp = *x;
  *x = *y;
  *y = temp;
}

Variable addresses must be used when calling the function:

int a = 2, b = 7;
swap(&a, &b);

References

C++ offers references as a neater alternative:

void swap(int& x, int& y)
{
  int temp = x;
  x = y;
  y = temp;
}
1
int& = reference to int
2
Cleaner syntax: no need to dereference using * 
int a = 2, b = 7;
swap(a, b);
3
Cleaner syntax: no need to take addresses using &

Note

Think of references as aliases for variables; they are a little like pointers but they can only point to existing variables and we cannot do pointer arithmetic with them

Using const References

  • Function parameters are copies of the arguments passed to the function – which is costly for large arguments such as vectors with many elements

  • Declaring a parameter as a reference avoids a potentially costly copy operation

  • Additionally declaring the parameter as const makes it clear that a reference is being used for efficiency, rather than to change the parameter

Example

Function to read data from a file into a vector:

void read_data(const string& fname, vector<float>& data)
{
  // fname is a const reference because it isn't changed;
  // data is not const because it is changed
  ...
}

Function to compute the mean of values in a vector:

float mean_value(const vector<float>& data)
{
  // data can be const because computing mean won't change it
  ...
}
